# 📈 自选股票助手

一个简洁实用的Chrome浏览器扩展，帮助你实时追踪自选股票行情和持仓收益。

## ✨ 功能特性

| 功能 | 说明 |
|------|------|
| 🔢 大盘指数 | 实时展示上证指数、沪深300、深证成指、创业板指 |
| 🔍 股票搜索 | 支持股票代码或名称模糊搜索 |
| 📊 持仓管理 | 添加/删除自选股票，记录持有股数和成本价 |
| 💰 收益计算 | 自动计算每只股票和总体的收益额、收益率 |
| 🔄 实时刷新 | 10秒自动刷新行情数据 |
| 🌙 暗色模式 | 支持标准/暗色主题一键切换 |

## 📦 安装方法

### 方式一：开发者模式安装（推荐）

1. 下载并解压本项目（或 `stock-assistant.zip`）
2. 打开Chrome浏览器，地址栏输入 `chrome://extensions/`
3. 右上角开启 **「开发者模式」**
4. 点击 **「加载已解压的扩展程序」**
5. 选择解压后的 `stock-assistant` 文件夹
6. 完成！扩展图标会出现在浏览器右上角

### 方式二：Chrome Web Store（暂未上架）

后续如上架商店，可直接搜索「自选股票助手」安装。

## 🚀 使用指南

### 添加股票
1. 点击浏览器右上角的扩展图标打开面板
2. 在「添加股票」搜索框输入股票代码（如 `600519`）或名称（如 `茅台`）
3. 从下拉结果中点击要添加的股票

### 设置持仓
1. 点击底部「编辑」按钮进入编辑模式
2. 输入持有股数和成本价
3. 点击「完成编辑」保存

### 查看收益
- **收益额** = (现价 - 成本价) × 股数
- **收益率** = (现价 - 成本价) / 成本价 × 100%
- 底部显示所有持仓的总收益汇总

### 其他功能
- 点击「行情中心」跳转东方财富行情页面
- 点击 🔄 按钮手动刷新数据
- 切换「暗色模式」适应不同使用环境

## 🔧 技术细节

- **Manifest Version**: V3
- **数据来源**: 东方财富公开行情接口
- **数据存储**: Chrome Storage API（本地持久化）
- **刷新间隔**: 10秒自动刷新
- **涨跌颜色**: 涨红跌绿（中国A股风格）

### API接口

| 用途 | 接口 |
|------|------|
| 股票搜索 | `searchapi.eastmoney.com/api/suggest/get` |
| 实时行情 | `push2.eastmoney.com/api/qt/stock/get` |
| 大盘指数 | `push2.eastmoney.com/api/qt/ulist.np/get` |

## ⚠️ 注意事项

1. 本扩展调用东方财富公开接口，仅供个人学习使用
2. 数据仅供参考，不构成投资建议
3. 交易时间外数据可能不更新
